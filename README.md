## Features

- Lightweight SIP softphone
- Supports Windows x32 & x64
- Highly customizable plugin architecture
  - Contacts book
  - Calls log
  - Hot keys support
  - OS-native click to call
  - Browser trigger on incoming call
  - Advanced SIP headers support for Call Centers
